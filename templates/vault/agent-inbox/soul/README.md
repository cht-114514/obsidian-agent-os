# agent-inbox/soul

Local soul store for Obsidian Agent OS (agent-owned; always human-readable).

| Path | Meaning |
|------|---------|
| `IDENTITY.md` | Who the agent is |
| `SOUL.md` | Voice, hard boundaries, default behavior |
| `profile.md` | Stable model of the user (after confirmed insights) |
| `style.md` | Communication style |
| `cares.md` | Proactive care policy |
| `insights/drafts/` | Insight drafts |
| `insights/accepted/` | Accepted insights |
| `feedback/` | 👍👎 logs |
| `thoughts/` | Optional thought archives |
| `pending-care.md` | Care strip queue |

These files are **yours to edit**. The open-source beta ships **empty templates** only — no author's personal persona.
