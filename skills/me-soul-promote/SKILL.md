# me-soul-promote

与 `me-digest` / `me-write-insight` 同级：Grok 扫描 → pending 计划 → Accept 写入。

## 用法

```
/me-soul-promote
```

可选补充说明，例如：`/me-soul-promote 优先处理沟通语气`

## Accept 语义

写入：
- `agent-inbox/soul/profile.md`
- `agent-inbox/soul/style.md`
- `agent-inbox/soul/SOUL.md`

IDENTITY 不自动改。
