# Notices & Credits

## Status

**Me.Soul is currently a public beta / test build**, not a finished product.  
APIs, vault layout, and UI may change. Use at your own risk; back up your vault.

## Creative inspiration — Cola

Me.Soul’s **soul loops** (思绪 / 心迹 / 牵挂 — thoughts, insights, proactive care) and the feel of a living co-founder agent are **inspired by Cola** (also referred to as KOLLA / ColaOS in community discussion).

We do **not** claim to be Cola, affiliate with Cola, or redistribute Cola’s proprietary code.  
This project re-implements related *ideas* for a **vault-native Obsidian + OpenClaw/Grok** stack:

| Idea (Cola-inspired) | Me.Soul landing |
|----------------------|-----------------|
| Observable “soul” side of the agent | `agent-inbox/soul/*` + UI blocks |
| 思绪 (thoughts) as marginalia | `:::thought` fences, default expanded |
| 心迹 (insights) about the user | `me-write-insight` → confirm → profile |
| 牵挂 (care) with guardrails | `cares.md` + `me-care-check` |

If you know Cola, please support their work; Me.Soul is an independent open-source experiment.

## Other lineage
